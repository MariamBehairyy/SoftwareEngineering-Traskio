Project Name: Traskio Task Manager
Description:
A web-based application for task management. Users can add, edit, delete, view tasks. Admins can assign tasks to users.

Functional Requirements:
1. User authentication and authorization
2. CRUD operations for tasks
3. Admin task assignment
4. View task list by user/project

Non-functional Requirements:
1. Web-based (Flask)
2. Database: SQLite
3. Simple and intuitive UI
4. Responsive and mobile-friendly

Technologies:
- Python 3.13
- Flask
- SQLite
- HTML/CSS/JS
